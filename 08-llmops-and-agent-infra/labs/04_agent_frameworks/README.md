# Lab 04: Agent 框架实战

## 目标
掌握主流 Agent 范式：ReAct、Plan-and-Execute、LangGraph 有向图编排。

## 前置准备
```bash
pip install langchain langchain-openai langgraph autogen-agentchat
```

## 实验内容
1. `react_agent.py` — ReAct Agent 手动实现 + LangChain 实现
2. `plan_execute_agent.py` — Plan-and-Execute Agent
3. `langgraph_agent.py` — LangGraph 有向图编排
4. `autogen_demo.py` — AutoGen 多 Agent 对话
