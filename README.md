# AI Infra 全栈实战体系

从算力底座到应用治理的完整 AI 基础设施学习与实战项目。

## 硬件环境

- 单节点 8x NVIDIA H20（96GB HBM3 / 卡，总计 768GB 显存）
- 适合推理优化、中等规模分布式训练、微调实验

## 每个模块的结构

```
module/
├── theory/          # 理论笔记（Markdown），知其然更知其所以然
├── labs/            # 动手实验（01-10），由浅入深
│   ├── 01_xxx/     # 每个 lab 独立可运行
│   └── ...
└── project/         # 企业级实战项目（学习规模，完整工程结构）
    ├── src/
    ├── tests/
    ├── Dockerfile
    └── README.md
```

## 使用方式

1. 按优先级选择模块
2. 先读 theory/ 建立理论基础
3. 按编号顺序做 labs/ 实验
4. 完成 project/ 作为实战项目
