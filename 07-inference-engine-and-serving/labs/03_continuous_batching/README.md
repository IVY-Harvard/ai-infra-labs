# Lab 03: Continuous Batching 对比

## 目标

- 模拟 Static Batching 和 Continuous Batching 的行为
- 量化对比两者的吞吐和延迟差异
- 理解 Continuous Batching 的优势来源

## 实验内容

1. **static_batching.py** — 静态批处理模拟
2. **continuous_batching.py** — 连续批处理模拟
3. **throughput_comparison.py** — 吞吐对比

## 运行方式

```bash
python static_batching.py        # 静态批处理模拟
python continuous_batching.py    # 连续批处理模拟
python throughput_comparison.py  # 对比分析
```
